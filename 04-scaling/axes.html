Axes

svg element -- group (path, group of ticks, text) The axis, ticks and values are created as a group. The axis line is given the class name "domain"; the ticks are created within a separate group, internal to the axis group, with each tick having the class name "tick" and each has a 'line' and 'text' attribute. The 'title' is a seperate text element.
